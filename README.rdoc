== README

Integrate Github notifications in a Chatwork room.

How to do:

* Request and get your own API Token from Chatwork.
* Create a Chatwork room with all members added and room ID issued.
* Clone repository into local.
* Deploy to personal Heroku.
* Add webhook to Github repository with payload URL from heroku at: https://github.com/{your-project}/settings/hooks/new

It must work !
